# ExpressApp

NodeJs Express RESTful API
Ensure you have nodejs >= 12 installed

## Run Application

Clone

- `git clone https://github.com/evamaina/ExpressApp` 

Install dependencies

- `npm install`

### Start

Live-Reload: 

- `npm run dev-start`

Node: 

- `npm start`

## Test

-  `npm test`
